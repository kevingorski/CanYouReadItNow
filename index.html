Can You Read It Now?

Analyze the readability of your text on the Web

Can You Read It Now (or CYRIN) provides the numerical basis for implementing features related to the visual readability (not the reading comprehension level) of text on the web. CYRIN powers the bookmarklet of the same name

  • Development Version
  • Production Version (minified)

What CYRIN provides

Right now, there are seven scores taken from eighteen metrics:

  • Font Size
  • Justification
  • Line Leading
  • Line Length
  • Margin & Padding
  • Percentage of Text Stylized
  • Text Color Contrast

Several of these scores are based off of formulas given in the W3C's Web Content Accessibility Guidelines.

Demonstration

Can You Read It Now is a jQuery plug-in, so it is invoked just like a jQuery function:

var analysis = jQuery('#Synopsis p').CYRIN();

> Run Example

In this example, a single paragraph is analyzed and a big ol' tree comes back. You can inspect the metrics that were gathered to come up with the scores that are returned, which helps with debugging the results.

More complicated scenarios work as expected too, like analyizing the body element of a page or using a selector that returns multiple items. The root analysis will be a weighted result of all elements provided and each child analysis describes both the aggregate scores of that entire tree and that specific node if there is any text to analyze.

Objects displayed using prettyPrint.js

Possible Uses

This type of analytics could be used for:

  • Suggesting visual readability improvements for any page in a browser (like the CYRIN bookmarklet)
  • Generating CSS to improve readability for a plugin like StyleBot or GreaseMonkey
  • Real-time analysis of theme or styling changes (in a blog or CMS administration site)
